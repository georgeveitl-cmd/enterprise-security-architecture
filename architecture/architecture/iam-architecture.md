# Identity Security Architecture

## Overview

Identity security is a critical component of enterprise cybersecurity. Organizations must manage both human and non-human identities while controlling access to critical systems.

## Core Components

IAM – Identity and Access Management  
IGA – Identity Governance and Administration  
PAM – Privileged Access Management  

## IAM Functions

• Authentication (SSO / MFA)  
• Identity lifecycle management  
• Access provisioning  
• Access reviews  

## Governance Controls

• Role-Based Access Control (RBAC)  
• Privileged access monitoring  
• Joiner/Mover/Leaver identity lifecycle  
• Service account governance
