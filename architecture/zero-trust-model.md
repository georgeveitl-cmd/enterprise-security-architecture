# Zero Trust Security Architecture

## Overview

Zero Trust is a security model that assumes no user or system should be automatically trusted, even if it originates from inside the network perimeter.

Access to resources must always be verified and continuously validated.

## Core Principles

• Verify explicitly  
• Use least privilege access  
• Assume breach  

## Security Components

Identity Provider  
Multi-Factor Authentication  
Conditional Access Policies  
Endpoint Security Monitoring  
Network Segmentation  
Continuous Security Monitoring

## Benefits

• Reduced attack surface  
• Improved identity verification  
• Stronger access control
